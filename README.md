# Lingmo OS Installer
This installation program is developed based on [Vanilla OS Installer](https://github.com/Vanilla-OS/vanilla-installer)

### Dependencies

- build-essential
- meson
- libadwaita-1-dev
- gettext
- desktop-file-utils
- libgnome-desktop-4-dev
- libgweather-4-dev
- python3-requests
- gir1.2-vte-3.91
- libnma-dev
- libnma-gtk4-dev
- mutter-common
- x11-xkb-utils
